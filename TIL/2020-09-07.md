Ch7_2 부품번호 찾기 문제

풀이 방법 3가지
1. 이진 탐색
- 부품 배열에서 하나씩 요소를 꺼내서  
이진 탐색 -> 부품이 존재하는지 확인

2. 계수 정렬
- 가게의 부품을 입력 받을 때마다 해당 인덱스에 1을 대입하고,    
요청한 부품 번호의 인덱스에 값이 0인지 1인지 확인

3. Set 특성 이용
- 가게의 부품을 미리 Set에 등록시켜 놓고,  
요청한 부품을 Set에 등록한 뒤 Size가 달라졌는지 확인  
-> Size가 달라졌다는 건 중복이 없다 = 부품이 없는 것   
-> Size가 똑같다는 것은 중복이 있다는 것 = 부품이 있다
